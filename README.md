# Ecommerce JSON Server Backend

## Overview

This project is a backend API built using JSON Server for the Urban Kicks Admin Dashboard application.

The backend provides product data for the React frontend application and supports CRUD operations including fetching, adding, editing, and deleting products.

---

## Technologies Used

- JSON Server
- Node.js

---

## Features

- REST API simulation
- Product data management
- Supports GET, POST, PATCH, and DELETE requests
- Hosted on Render for public access

---

## API Endpoints

### Products

```txt
/products
```

### Store Information

```txt
/store_info
```

---

Install dependencies:
